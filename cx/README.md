# cx

`cx` is a full-access cmux-oriented pisolate. It installs:

- `pi-cx` for interactive cmux work.
- `p-cx` for one-shot cmux work.

The recipe defaults to allowing every CLI command, including all `cmux` commands and shell composition. Pisolates still provides a stable launcher, isolated working directory, isolated Pi session state, and recipe-specific prompt, but it is not a privacy sandbox.
